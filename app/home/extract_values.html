Help: Extract Values

Per default PrepJet starts extracting at the first position and ends extracting at the second position of the selected delimiter from the left. You can specify the count of the delimiter (1, 2, ...) as well as the direction from which to count if you want a different setting. The extracted content will be inserted in a separate column.
