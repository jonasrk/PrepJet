Add Column

Add data from another worksheet to your table.
Example: You have a table with names, surnames and birth dates of your customers. You want to add the e-mail addresses from a delivery information table based on the name, surname, and birthdate.
Select
1. The tables to be used (customer table and the delivery information table)
2. The column(s) to be added to the first table (column that contains the e-mail adresses)
3. The match criteria for the data to be added in both tables (name, surname, and birthdates)

Split Values

Split the entries in one column into separate columns.
Example: you have a column that contains name, surname and birthday, each separated by a space. You want to have the birthdate in a separate column.
1. Select the column to be split (e.g. the column containing name, surname, and birthday)
2. Specify the delimiter, i.e. the symbol at which the column entry shall be split (e.g. whitespace)
3. Open the advanced settings [delimitor position] only if you do not want the entry to be split at the first delimitor (e.g. second space)

Extract Values.

Extract a part from a column entry based on specified delimiters
Example: you have a column that contains name, surname and birthday, each separated by a space. You want to extract the surname and write it to another column.
1. Select the column from which the part should be extracted (e.g. the column containing name, surname and birthday)
2. Specify the start and the end of the part to be extracted (e.g. first whitespace, second whitespace)
3.Select whether the delimitor (e.g. the whitespace in front of and after the surname) shall be included in your new column.

Trim Spaces

Remove leading and trailing spaces.

Detect Duplicates

Find and highlight duplicate data records
Example: You have a list of products with the features „category“ and „color“ and you want to find all duplicates based on product name and category.
1. Select the column(s) that shall be included in the duplicate comparison (e.g. product name and category)
2. Specify the match type: exact vs. Fuzzy. Fuzzy means that the data records can differ slightly (e.g. „T-Shirt“ vs. „Tshirt“)

Detect Outlier

Find and highlight that deviate from the overall pattern in the column (statistical outliers). This function only works with numeric values (i.e. columns with which you can do mathematical operations)
Example: You have a list of product prices and want to detect entries that could be flaws.
Select the column to detect the outlier and apply.

Validation Rule

Create rules to validate your data and to detect inconsistencies based on domain specific conditions.
Example: You have a price information table and you know that there are certain price ranges for different product types.
