PrepJet helps you to get your data ready for analysis - fast and simple.
You can easily shape your data like you need it.
Only prerequisite: each column needs a header.
This 60 seconds intro helps you to get started with PrepJet.

Add data from another worksheet to your table. Simply select
1. The tables to be used
2. The column(s) to be added to the first table
3. The match criteria in both tables

Split the entries from one column into separate ones
1. Select the column to be split
2. Specify the delimiter, i.e. the symbol at which to split
3. Put a cetain count of the delimiter in advanced settings if you do not want to split the data at each one of the delimiters

Extract a part from a column entry based on specified delimiters.
1. Select the column to extract from
2. Specify start and end
3. Select whether the delimiter should be included or not

Delete leading and trailing spaces
Simply select the applicable columns.

Find and highlight duplicate data records.
1. Select the column(s) to include in comparison
2. Specify match type (exact or fuzzy)
Fuzzy means the data records can differ slightly (e.g. „John Smith“ vs „John S. Smith“)

Find and highlight values that do not comply with the variance of the overall column.
Simply select the column to detect outlier in.

Check if your data complies to certain rules.
1. Select conditional column
2. Select operator (e.g. <; >; =)
3. Enter threshold or condition
4. Select dependend column
5. Select operator.
6. Enter condition
