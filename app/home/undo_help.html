Undo Help

MS Excel does in general not support "undo" for plug in operations. This applies also for the familiar shortcut ctrl + Z.


The regular Excel Undo function is allows you to undo operations up to the last interaction with a plug in. Once you use a plug in functionality undo is resetted.


PrepJet offers its own "undo" function so that you can go back to a previous state in case the result is not what you expected. However font and background colors will not be resetted.
